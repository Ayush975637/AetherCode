## ⚙️ Setup

### 1. Clone the repository

git clone https://github.com/your-username/your-repo.git
cd your-repo

### 2.Frontend Setup
cd frontend
npm install
npm run dev
  
###Create .env:
VITE_API_URL=http://localhost:3003

### 3.Backend Setup
cd backend
npm install
npm run dev

###Create .env:

PORT=3003
MONGODB_URI=your_mongodb_url
CLIENT_URL=http://localhost:5173

### 4.🚀 Deployment
Frontend → Vercel
Backend → Render
